## Customer Profile List

- Displays a list of all customer profiles.
- Use the **Create New Profile** button at the top-right to add a new customer.
- Each row includes an **Edit** button to update customer details.

---

![Favicon Image](/images/customerlist.png)

## Create New Customer Profile

To create a new customer profile:

1. Click the **Create New** button from the top-right corner of the list screen.
2. Fill in the basic details:
   - **Name**
   - **Email**
   - **Phone Number**
   - **Vendor**
   - Other necessary fields
3. Click **Save** to create the profile.

---

![Favicon Image](/images/newcustomer.png)

## Edit Customer Profile

- From the list view, click the **Edit** button next to any customer.
- Update the required fields.
- Click **Update** to save changes.

## Customer Detail Screen

Displays all the information related to a selected customer.

Below the main customer details, you'll find several tabs:

![Favicon Image](/images/detailcustomer.png)

### Orders

- Lists all orders related to the customer.
- Orders can also be edited from this tab.
  ![Favicon Image](/images/detailcustomer.png)

### Address

- Add or edit the customer's addresses.

![Favicon Image](/images/address.png)

### References

- View or manage references linked to the customer.

![Favicon Image](/images/refrences.png)

### Notes

- Add any relevant notes for internal use or record-keeping.
