title: "Dashboard"
---

The **Dashboard** provides a summary of the profile, offering key insights into orders and vendor activity.

## Overview

The Dashboard displays:

- **Order Count**: Total number of orders associated with the profile.
- **Vendor Order Count**: Number of vendor-specific orders linked to the profile.

This summary allows users to quickly understand their order activity and manage vendors effectively.

## Example

> Example:
>
> - Orders: `128`
> - Vendor Orders: `45`

## Navigation

To access the Dashboard:

1. Login to the application.
2. Navigate to the **Dashboard** section via the sidebar or top navigation.
