## Pickup Section

The **Pickup** section lists all orders received from the warehouse and provides key information about each one.

Each pickup entry includes:

- **Process Id** – It shows Process Id
- **Pickup Officer** – The assigned personnel handling the pickup.
- **Order ID** – Unique identifier for the order.
- **Pickup Confirmed** – Shows whether the pickup has been confirmed.
  ![Favicon Image](/images/pickuplist.png)

### Actions

You can click the **Actions** button to view detailed information about the pickup, including:

- Item list
- Pickup time and location
- Officer remarks
- Confirmation status
