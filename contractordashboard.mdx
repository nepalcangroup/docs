The **Contractor Dashboard** provides a summary of the contractor's profile, offering key insights into pickup orders and delivery activity.

## Overview

The dashboard includes:

- **Pickup Count** – Displays the total number of pickups associated with the contractor's profile.
- **Delivery Count** – Shows all the orders that are scheduled for delivery.
